## 📌 Overview

Smart Outfit AI is an intelligent outfit recommendation system built using Python and Flask.  
It uses AI (Gemini API) to generate personalized fashion suggestions based on user inputs like gender, occasion, and budget.

---

## ✨ Features

- 👚 AI-based outfit recommendations  
- 🎯 Personalized suggestions  
- 💰 Budget-based filtering  
- 🛒 Shopping links  
- 🌐 Simple UI  

---

## 🛠️ Technologies Used

- Python  
- Flask  
- Gemini API  

---

## ▶️ How to Run the Project

1. Install dependencies:
   pip install flask google-generativeai

2. Run the application:
   python app.py

3. Open in browser:
   http://127.0.0.1:5000/
