# 👗 Smart Outfit AI

Smart Outfit AI is an intelligent outfit recommendation system built using **Python and Flask**.  
It uses **Google Gemini AI API** to generate personalized fashion suggestions based on user inputs like gender, occasion, weather, and budget.

---

## ✨ Features
- 👚 AI-based outfit recommendations using Gemini AI
- 🎯 Personalized suggestions based on user preferences
- 💰 Budget-based filtering system
- 🛒 Shopping links for recommended outfits
- 🌐 Simple and user-friendly web interface

---

## 🛠️ Technologies Used
- Python
- Flask
- HTML / CSS / JavaScript
- Google Gemini API
